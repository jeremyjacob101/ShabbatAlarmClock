# Shabbat Alarm Clock (iOS)

Simple iOS app for creating and managing Shabbat-related alarms.

## Requirements

- macOS with Xcode installed
- iOS Simulator or iPhone device

## Run

1. Open `ShabbatAlarmClockiOS.xcodeproj` in Xcode.
2. Select an iOS Simulator (or connected device).
3. Press **Run**.

## Project Structure

- `ShabbatAlarmClockiOS/Models` - Alarm data models
- `ShabbatAlarmClockiOS/ViewModels` - App view models
- `ShabbatAlarmClockiOS/Views` - SwiftUI screens/components
- `ShabbatAlarmClockiOS/Services` - App services
- `ShabbatAlarmClockiOS/Persistence` - Data storage/repository
